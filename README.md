# Bro Chat 浏览器扩展

## 项目概述
Bro Chat 是一个浏览器扩展，旨在为多个AI平台（包括Yuanbao和Gemini）提供统一的消息发送接口。该扩展通过模拟用户交互，实现了在不同AI平台网页上的自动化消息发送功能。

## 主要功能
- 支持多平台：兼容Yuanbao、Gemini等多个AI平台
- 自动化交互：模拟用户输入和点击操作
- 消息调度：提供统一的消息发送接口
- 环境检测：自动识别当前页面是否为支持的AI平台
- 日志记录：详细的运行日志输出，便于调试和问题排查

## 技术特点
- 使用CSS选择器和XPath混合策略定位页面元素
- 实现了优雅的错误处理机制
- 支持异步消息传递
- 提供重试机制确保消息可靠发送
- 采用模块化设计，代码结构清晰

## 文件结构
- `contentScripts/`：包含各个AI平台的适配脚本
- `popup/`：扩展弹出界面相关文件
- `funcs/`：功能模块和工具函数
- `backgroudtask/`：后台任务处理
- `icons/`：扩展图标资源

## 安装使用
1. 克隆仓库
2. 在浏览器扩展管理页面加载解压的扩展
3. 访问支持的AI平台页面即可自动激活
