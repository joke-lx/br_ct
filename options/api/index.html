API 设置

配置智谱 AI 的 API 信息，用于划词翻译和 OCR 功能。

智谱 AI API 的基础地址，通常为 https://open.bigmodel.cn/api/paas/v4

在智谱 AI 开放平台获取您的 API Key

API Key 将安全存储在浏览器本地，不会被上传到任何服务器

推荐使用 glm-4.5v 进行 OCR 和文本处理

使用说明

  • 访问 智谱 AI 开放平台 注册并获取 API Key
  • API Key 格式示例：1234567890abcdef.EdkVfX8wE0JtcZpP
  • 推荐模型：glm-4.5v（支持视觉和文本处理）
  • 配置完成后点击"测试连接"验证配置是否正确
  • 配置将保存在浏览器本地，不同设备需要分别配置
